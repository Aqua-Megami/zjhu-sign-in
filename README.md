# 湖州师范大学信息工程学院远程签到器

由于**湖师信工微频道**中新启用的**AI辅导员**功能几乎没有校验，因此我们可以很轻松地远程签到

## Run directly

- Download from **[Releases](https://github.com/Aqua-Megami/zjhu-sign-in/releases/latest)**
- Run `main.exe`

## Build from Source

### FrontEnd

- Install requirements

  ```bash
  npm i
  ```

- Development run

  ```bash
  npm run dev
  ```

- Build

  ```bash
  npm run build
  ```

### ServerSide
- Install `Python` (`3.11+` recommended)
- Install requirements
  ```bash
  pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
  ```
- Run directly
  ```bash
  python main.py
  ```
- Or pack with pyinstaller
  ```bash
  pyinstaller main.py
  ```

## Caution

- Notice that if you build before running, you should put the `dist` folder generated by npm to the dir of `main.exe` or `main.py`. The Flask would route `/page` to it.
- However, if you run in development envirionment, `vite` would make a proxy of the server-side. So you should visit the server of `npm`.
